Selected Research Projects

LACE: Latent-Adaptive Controllable Encoder

Learned Image Compression · Variable-Rate Control · PyTorch

Designed a controllable variable-rate image compression framework that extends pretrained single-rate learned compression models toward user-specified bitrate control.

  • Reused the strength of pretrained single-rate encoders while adding rate-aware latent modulation.
  • Developed an image-guided rate conditioning converter to generate content-adaptive control signals.
  • Focused on reducing model redundancy and improving deployment flexibility across bitrates.
PyTorch CompressAI Rate Control Image Compression

RGB-D Image Processing and 3D Coordinate Alignment

Computer Vision · Intel RealSense SDK · C++

Built image processing and 3D spatial alignment workflows for RGB-D sensing applications, including coordinate transformation and depth-related system integration.

  • Implemented C++ modules for camera-based 3D coordinate handling.
  • Integrated depth sensing pipelines using Intel RealSense SDK.
  • Connected algorithm design with practical system-level constraints.
